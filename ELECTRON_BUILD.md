# Сборка нативного приложения HARDY для Windows

## Требования
- Node.js 18+
- Git

## Инструкция по сборке

### 1. Экспортируй проект в GitHub
В Lovable: **Settings → GitHub → Export to GitHub**

### 2. Склонируй репозиторий
```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
cd YOUR_REPO
```

### 3. Установи зависимости
```bash
npm install
```

### 4. Собери веб-приложение
```bash
npm run build
```

### 5. Собери Windows приложение
```bash
npx electron-builder --win
```

### 6. Готово!
Файлы появятся в папке `release/`:
- `HARDY Кодексы-X.X.X-Setup.exe` — установщик
- `HARDY Кодексы-X.X.X-Portable.exe` — портативная версия

---

## Глобальные горячие клавиши

После установки приложения будет работать **глобальная горячая клавиша**:

| Комбинация | Действие |
|------------|----------|
| `Ctrl+Shift+H` | Открыть HARDY и юридического помощника |

> ⚠️ Используется `Ctrl+Shift+H` вместо `Ctrl+F`, чтобы не конфликтовать с другими программами.

---

## Трей (системный лоток)

Приложение сворачивается в трей при закрытии:
- Клик по иконке — открыть приложение
- Правый клик — меню с опциями

---

## Для разработки

Запуск в режиме разработки:
```bash
# Терминал 1: запустить веб-сервер
npm run dev

# Терминал 2: запустить Electron
npx electron electron/main.js
```
